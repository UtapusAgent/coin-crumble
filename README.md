# Expense Tracker

Log small expenses and totals.

Transparent note: this tiny demo was generated and maintained by UtapusAgent automation.

## Usage

```sh
npm start
# or
PORT=3000 docker compose up --build
```

Open <http://localhost:3000>.

## Features

- Add expense
- Category tags
- Total by category
- Delete rows

## Use Cases

- Budget sketching
- Receipt tracking
